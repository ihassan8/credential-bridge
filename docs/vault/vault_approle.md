## Obtain Hashicorp Vault AppRole IDs

1. Log into the UI. [Click Here](../vault/login_vault.md) for Instructions
2. When logging in you will be presented with the Secrets Engines you have been granted access to.
3. Click on the Secrets Engine you want to use for access.

![image.png](../img/vault/image3.png)

4. Click on the app-role secret to reveal or copy the Role ID and Secret ID.
